# Awesome the awesome.

Install
-------

```
composer install
cp config.yml.dist config.yml
vim config.yml # and set your credentials
```

> And if your password is 0000, don't forget to set it as string in the config

Run
---

```
php -S localhost:8000
```

Access it at http://localhost:8000
